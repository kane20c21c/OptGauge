# OptGauge 해석 노트 — 지표를 읽을 때 실수하지 않기 위한 기록

> Layer C 서술 규칙의 원천. 실제 데이터에서 발견한 함정·사례를 누적한다.
> 원칙: 지표는 방향 예측이 아니라 자세(posture) 기술. 단일 해석 단정 금지.

## 함정 1 — 만기 리셋을 수요 변화로 오독 (2026-07-16 실증, 클로이 실수 사례)

- 관찰: PCR_OI 롤60 7%ile, OI 급감 → "방어 포지션이 풀리는 중"으로 서술.
- 실제: 7/9 만기일에 OI 718,104 → 360,213 반토막 = **7월물 소멸의 기계적 리셋**.
  이후 신월물 재구축 중(→425k). 수요 철수가 아니라 보험 갱신 국면.
- 규칙: **OI·PCR 해석 전에 만기 캘린더부터 확인.** 만기 전후 ±5거래일의
  OI 계열 변화는 기술적 왜곡 후보를 반드시 병기.

## 함정 2 — 스큐 축소 = "급락 가능성을 낮게 본다"로 직행 (2026-07-17 Kane 질문)

체인 "스큐 축소 → 하방 보험 수요 감소 → 급락 확률 하향"은 **후보 중 하나일 뿐**.
스큐는 풋IV−콜IV 의 차이라서, 축소되는 경로가 최소 4가지:

| 경로 | 의미 | 판별 힌트 |
|---|---|---|
| 풋IV 하락 주도 | 하방 불안 완화 (Kane 체인 성립) | 풋IV↓ + IV 전체↓ + 지수 안정 |
| **콜IV 상승 주도** | 상방 추격/숏커버 — 하방 인식 불변 | 콜IV↑ + 지수 급등 중 (실측: 6/12, 7/1~2 역전) |
| 풋 공급 증가 | 변동성 매도 유행 — 위험 인식과 무관 | 풋IV↓ + 풋 OI↑ (매도 적체) |
| 기계적 평탄화 | 고IV 레짐에서 스마일 자체가 눕는 효과 | IV 전체 급등 동반 |

- 규칙: **스큐 변화는 반드시 "어느 쪽이 움직였나"로 분해해서 읽는다.**
  → Layer A 확장 후보: ΔIV_put(0.5σ) / ΔIV_call(0.5σ) 분리 저장 (스큐 귀속 분해).

## 함정 3 — 보험료를 객관 확률로 읽기

- 옵션 가격 = 위험중립 확률 × 위험 프리미엄 + 수급. "시장이 보는 확률"이 아니라
  "시장이 지불하는 가격". 보험을 안 사는 이유가 위험이 없어서가 아닐 수 있다:
  헤지 예산 소진, 기초 포지션 자체를 줄임(오히려 약세 행동), 보험료가 비싸서 포기.
- 참고: 스큐의 급락 예측력은 학계에서도 논쟁적 — 예측 지표가 아니라 상태 지표로 쓴다.

## 함정 4 — 갭/연휴/신규상장 기술 왜곡 (구현에 반영됨)

- 추석 연휴(달력 8일)를 수집 갭으로 오판 → 롤링 리셋 (GAP 기준 12일로 수정, 069f87d).
- 급등락 직후 ATM 부근 행사가는 신규 상장이라 OI 극소가 정상.
- RV 계산은 반드시 연속 지수 시계열로 (옵션 수집 갭 오염, 62f8dcf).
