## 0. 공통 규칙

**U0-1. 대상**: `Underlying == "코스피200 옵션"` (월물). 미니·위클리·코스닥150 은 v0.1 제외
(v2 후보 — 위클리는 단기 이벤트 게이지로 별도 가치).

**U0-2. 주간 세션만**: `Name` 에 `"(야간)"` 미포함 행만 사용.
검증 게이트: 주간 행 IV>0 비율 확인 (2026-06-15 실측: 주간=IV 전량 유효, 야간=IV 0).

**U0-3. 만기 파싱**: `Name` 형식 `"코스피200 C 202607   850.0"` → 만기 YYYYMM, 행사가 float.
파싱 실패 행은 제외하되 일별 실패율 >1% 면 경고 (코드체계 개편 등 구조 변화 감지).

**U0-4. 만기일·롤 규칙**: 만기 = 매월 둘째 목요일 (거래일 캘린더로 보정).
근월물 = 잔존 거래일 ≥ 5 인 최소 만기. 잔존 < 5 거래일이면 차월물로 롤
(만기주 감마 왜곡 배제). 차월물 = 근월물 다음 만기.

**U0-5. 기초자산 가격 S_t**: LLV KOSPI200 parquet 의 당일 종가. 옵션 Date 와 지수 Date 정합
필수 (불일치 시 해당일 게이지 결측 처리 — 가짜 값 생성 금지).

**U0-6. IV 유효성**: IV ≤ 0 또는 IV > 300(%) 행 제외. ATM 산출에 필요한 행이 없으면
해당 지표는 그날 **결측(NaN)** — 인접값 보간 금지 (Kane 엄격 정책: 부분 데이터로 가짜 값 금지).

**U0-7. 표기**: IV·스큐는 %p 소수 둘째 자리, 백분위는 정수, 비율은 소수 둘째 자리.

## 1. G1 — IV 수준·변화

- **ATM 행사가** K_atm = argmin |K − S_t| (근월물, 콜·풋 공통)
- **ATM_IV_t** = ( IV_call(K_atm) + IV_put(K_atm) ) / 2
- **ΔATM_IV_t** = ATM_IV_t − ATM_IV_{t−1} (롤 발생일은 결측 — 월물 불연속)
- **RV20_t** = stdev( ln(S/S_prev), 20거래일 ) × √252 × 100
- **VRP_t** = ATM_IV_t − RV20_t  (변동성 리스크 프리미엄 — 음전환이 이상 신호 후보.
  단, 쇼크 후행 음전환은 잔상 후보 — 해석노트 함정 7 의 선행/후행 구분 적용)
- **[확정 2026-07-18] 보조 지표 RV_fast_t** = √( EWMA_{λ=0.90}( r² ) ) × √252 × 100,
  r = ln(S/S_prev), RV20 과 동일한 **연속 지수 시계열** 사용 (함정 4).
  구현: RiskMetrics 제로평균 · 유한표본 정규화 가중(pandas ewm adjust=True) · 최소 20일.
  **VRP_fast_t** = ATM_IV_t − RV_fast_t.
  용도 = **조기경보 전용** (선행 음전환을 RV20 대비 5~8일 조기 포착 — 2020 실측 1/20 vs 1/28).
  레벨·블록 정본은 RV20/VRP 유지 — 5벌 실증(2020·2026 에피소드, λ=0.97/0.94/0.90/RV10):
  | 변형 | 진입지연(2020) | 음수블록(2020) | 평온기 음전환 진입/년 |
  |---|---|---|---|
  | RV20 (정본) | 3일 | 26일 | 6.2 |
  | EWMA λ=0.94 | 3일 | **54일 (꼬리 악화)** | 7.2 |
  | **EWMA λ=0.90 (보조)** | **1일** | 45일 | 8.0 |
  | RV10 | 1일 | 1일 (블록 조각남) | 7.8 |
  근거: 지수 감쇠는 컷오프가 없어 쇼크 꼬리가 길어짐 (EWMA 역설) — 균등창 하드 컷오프가
  꼬리 절단 장치. λ=0.90 은 평균 가중 연령 9.0일로 균등창20(9.5일)과 동급이면서 최신
  충격 반응만 빠름. 상세 실증 = 해석노트 함정 7, output/vrp_weighting_variants.html

## 2. G2 — 스큐

- K_put = argmin |K − 0.90·S_t| (근월물 풋), K_call = argmin |K − 1.10·S_t| (근월물 콜)
- **Skew_raw_t** = IV_put(K_put) − IV_call(K_call)
- **Skew_norm_t** = Skew_raw_t / ATM_IV_t  (레짐 간 비교용 — IV 80% 레짐과 15% 레짐 직접 비교 불가)
- 선택 행사가가 0.85~0.95 / 1.05~1.15 밴드 밖이면 결측 (호가 공백 방지)
- **머니니스 폭의 트레이드오프** (2026-07-16 Kane 논의): 좁히면(0.95/1.05) 유동성·신뢰도 ↑
  꼬리 정보 ↓ (ATM IV 와 상관 ↑ = 독립 정보량 ↓), 넓히면(0.85/1.15) 꼬리 민감도 ↑ 노이즈·결측 ↑.
  더 근본 문제: 고정 머니니스는 레짐에 따라 σ-거리가 변함 — 1개월 만기 10% OTM 이
  IV 15% 레짐에선 ~2.3σ(깊은 꼬리), IV 80% 레짐(2026)에선 ~0.4σ(준ATM).
  → 시계열 비교가능성 훼손. 보정안 = **vol-조정 머니니스**: K = S·exp(±k·σ_ATM·√T), k=1.
- **[확정 2026-07-16] 스큐 정본 = vol-조정 ±0.5σ 최근접 스냅** (`Skew_vol05s`, 별칭 `Skew`).
  5벌 실증 비교(948일: 코로나/평온기/2026 극단 레짐) 결과:
  | 변형 | 레짐 일관성(2026/코로나) | 거칠기(2026) | NaN율(2026) |
  |---|---|---|---|
  | 0.90/1.10 고정 | 0.61 | 3.53 | 9.4% |
  | 0.95/1.05 고정 | 0.38 | 4.75 | 0.0% |
  | ±1σ 스냅 | 1.28 | 4.41 | **43.8% → 탈락** (콜 타깃이 상장 범위 밖) |
  | **±0.5σ 스냅 (정본)** | **0.85** | **3.35** | 6.2% |
  | ±0.5σ 보간 | 0.88 | 3.63 | 15.6% (외삽 금지로 엣지 결측 ↑ — 기각) |
  보조 지표 = 0.95/1.05 고정 (NaN 0%, 유동성 최상 — 크로스체크용).
  참고: 2026 거칠기 3.3~4.8%p 는 상당 부분 실제 스마일 재가격 (방법론 노이즈 아님).

## 3. G3 — 기간구조

- **TS_diff_t** = ATM_IV(차월) − ATM_IV(근월)  (음수 = 백워데이션 = 단기 스트레스)
- **TS_ratio_t** = ATM_IV(차월) / ATM_IV(근월)
- 보조 (참고 전용, 플래그 제외): VKOSPI 선물 커브 (KIS A04 월물 — OI 극소, 호가성 데이터)

## 4. G4 — 미결제 분포

전월물 합산(A)과 근월물(B) 두 벌 계산:
- **PCR_OI_t** = Σ OI_put / Σ OI_call
- **OI_center_call/put_t** = Σ(K·OI) / Σ(OI) — OI 가중 중심 행사가. 게이지는 (OI_center − S_t)/S_t
- **OI_conc_t** = 상위 5개 행사가 OI 합 / 전체 OI (콜·풋 별도)
- **ΔOI_total_t** = 전체 OI 합의 전일 대비 변화율
- ⚠ 신규 상장 행사가는 OI 극소가 정상 (급등락 직후 ATM 부근이 신규일 수 있음 —
  2026-07-16 새벽 관측: ATM OI 40계약). OI_center·conc 는 이 왜곡에 상대적으로 강건.

## 5. G5 — VKOSPI

- **VK_t** = VKOSPI 종가 (LLV parquet), 실시간은 fetch_vkospi_realtime (KIS 0503)
- **ΔVK_t**, **VK_basis_t** = VK_t − ATM_IV_t (모델프리 vs ATM 괴리 — 스마일 정보)

## 6. Layer B — 정규화·플래그 (전 지표 공통)

각 지표 x_t 에 3벌 병기:
1. **P_full**: 전체기간 백분위 (역사적 절대 위치 — 포화 자체도 정보)
2. **P_roll60**: 롤링 60거래일 백분위 (현 레짐 내 상대 위치 — 일일 보고 주력)
3. **Z_delta**: Δx_t 의 60일 z-score (변화 속도 이상)

플래그 (초안, 실증 후 조정): P_roll60 ≥ 95 → HIGH / ≤ 5 → LOW / |Z_delta| ≥ 2.5 → JUMP.
복합 플래그는 v0.2 에서 (예: IV JUMP + Skew HIGH + PCR HIGH 동시 = 방어수요 클러스터).

**2026 레짐 주의** (Kane 확인): VKOSPI 2026-02-26 이후 50 미만 없음 — P_full 은
2026년 내내 90%ile+ 포화 예상. 해석의 주축은 P_roll60·Z_delta. 실증 비교(2단계 3번)에서
윈도(60 vs 120 vs 250) 확정.
